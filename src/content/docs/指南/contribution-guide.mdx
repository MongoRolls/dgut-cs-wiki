## 项目启动

<Steps>

1. 安装必要的开发环境

   - 安装 Node.js
   - 安装 pnpm

2. 安装项目依赖并启动开发服务器

   ```bash
   pnpm install
   pnpm dev
   ```

   执行完上述命令后即可在本地启动项目

3. 构建项目
   ```bash
   pnpm build
   ```
   此命令会检查项目语法错误并生成生产环境代码

</Steps>

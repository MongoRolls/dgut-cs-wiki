# 知识分享:计算机扫盲

## 背景
> 对于很多大学生来说，计算机知识方面是个盲区，甚至上了大学才第一次用电脑，会接触到很多陌生的名词让人无从下手，我也不例外，本文章作为一个引子，粗略介绍计算机知识中各种专有名词。
